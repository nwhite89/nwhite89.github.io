The objective of this Facebook application was to create a web interface for Mentos UK. Which were launching a new competition called Shake Your Money Maker.

The site will allow competition participants to enter their codes found on various different Mentos product wrappers, and play a Flash game in which you shake the product using a webcam to increase a power bar.

To complete this project it was created using CodeIgniter, Compass, GruntJs and CoffeeScript.

For this project my work was mainly working on the front-end CSS using Compass and the back-end CodeIgniter PHP. With some work with CoffeeScript where needed.
