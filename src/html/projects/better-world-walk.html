The aim of this project was to provide employees within Ethicon EMEA (J&J) access to view information about an internal fund raising exercise. A number of Ethicon employees collectively are walking to South Africa (using pedometers).

The site will allow participants to enter their amount of foot steps which will equal Kilometres. Depending on the amount of Kilometres entered, the background will change stating where they currently are going from Paris to South Africa.

The site was created using the Yii Framework using Twitter Bootstrap for the mobile site. On the Desktop site, the home page has been created using Scrollorama allowing the user to scroll down the page and basically walk down a road providing more information of the site.

My work on this project was purely as developer, working on front-end JavaScript/jQuery with work on the Parallax of the site. Whilst also working on the server-side PHP for such things as the events, and a Cronjob run every hour running a simple page scraper to update how much had been raised whilst looking at the OperationSmile site. Other than this I worked on the mobile site myself working with Twitter Bootstrap but run as a complete different Yii theme rather than a responsive site.
