The SmartTouch website was created to provide further information on BioSense Webster's product. The site was created using Twitter Bootstrap.

It is a Repsonsive layout website which has been created as a single paged website, allowing the user to scroll (or swipe) down the page.

For this project I worked as Developer lead, and created the Front-end HTML, CSS and jQuery/JavaScript on the site. The server-side code was outsourced to be created on Drupal.
