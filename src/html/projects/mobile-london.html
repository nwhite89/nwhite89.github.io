This project was completed for my dissertation which I recieved a first. My finished dissertation document can be found here in PDF format. Mobile London was created with Node.js, Express, HTML5, jQuery and jQuery Mobile; which provides users easy access to information provided by Transport For London (TFL) while travelling around London via public transport whilst using a smartphone or tablet device.

The web application was created to work on all mobile devices which support HTML5 however, optimised for iOS.
