Scoota's Rig Creative toolkit, is built using Angular, Webpack and Django. It allows creative designers to upload their PSD's straight to the toolkit in order to create Rich media ads that can be served through the Rig Distribute platform.

Once a user has uploaded their PSD and the creative has been created, using the UI the user can add Video's, HTML, CSS, JS and incoporate actions to create a imersive experience for the end-user.
